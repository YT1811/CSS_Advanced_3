# Spillverden - CSS Animasjonsprosjekt

Et interaktivt nettside-prosjekt som demonstrerer bruk av CSS-animasjoner, transitions og transforms.

## Innhold

- `index.html` - Hovedside med tre spillkategorier
- `style.css` - All styling og animasjoner

## Funksjoner

### Animasjoner (@keyframes)
- Glødende header med pulserende effekt
- Flytende ikoner
- Hoppende boks i footer
- Roterende stjerne

### Transitions
- Kort som løfter seg ved hover
- Bilder som zoomer og roterer
- Knapper og lenker med smooth overganger

### Transforms
- Skjeve overskrifter (skew)
- Skalering og rotasjon av bilder
- Translatering av elementer

## Hvordan bruke

1. Last ned begge filene
2. Åpne `index.html` i en nettleser
3. Hover over elementer for å se animasjonene

## Teknologier

- HTML5
- CSS3 (Animations, Transitions, Transforms)
